# Быстрый старт на телефоне

## 📱 Шаг 1: Установите Expo Go

### Android:
1. Откройте Google Play Store
2. Найдите "Expo Go"
3. Установите приложение

### iOS:
1. Откройте App Store
2. Найдите "Expo Go"
3. Установите приложение

---

## 🔄 Шаг 2: Перезапустите Expo сервер

### В терминале (где запущен Expo):

1. **Остановите сервер:**
   - Нажмите `Ctrl + C`

2. **Запустите заново:**
   ```bash
   npx expo start --clear
   ```

3. **Дождитесь появления меню:**
   ```
   › Metro waiting on exp://192.168.x.x:8081
   › Scan the QR code above with Expo Go (Android) or the Camera app (iOS)
   
   › Press a │ open Android
   › Press i │ open iOS simulator
   › Press w │ open web
   ```

---

## 📲 Шаг 3: Подключите телефон

### Вариант A: QR-код (самый простой)

**Android:**
1. Откройте приложение **Expo Go** на телефоне
2. Нажмите **"Scan QR code"**
3. Отсканируйте QR-код из терминала
4. Приложение загрузится автоматически

**iOS:**
1. Откройте приложение **Камера** на iPhone
2. Наведите на QR-код из терминала
3. Нажмите на уведомление
4. Откроется Expo Go и загрузится приложение

### Вариант B: Через команду в терминале

**Android:**
- В терминале Expo нажмите **`a`**
- Приложение откроется на подключенном Android устройстве

**iOS:**
- В терминале Expo нажмите **`i`**
- Приложение откроется на подключенном iOS устройстве

---

## ⚠️ Важно: Телефон и компьютер должны быть в одной Wi-Fi сети!

Если не работает:
1. Убедитесь, что телефон и компьютер подключены к одной Wi-Fi сети
2. Проверьте, что файрвол не блокирует порт 8081
3. Попробуйте перезапустить Wi-Fi на обоих устройствах

---

## 🔧 Если не работает:

### Проблема: "Unable to connect to Metro"
**Решение:**
1. Убедитесь, что Metro сервер запущен
2. Проверьте, что порт 8081 не занят
3. Попробуйте: `API_URL="https://<ngrok>.ngrok-free.app" API_KEY="superdev123" npx expo start --clear --tunnel` (использует HTTPS туннель)

### Проблема: QR-код не сканируется
**Решение:**
1. Увеличьте размер терминала, чтобы QR-код был четче
2. Или используйте команду `a` (Android) / `i` (iOS) в терминале

### Проблема: "Network request failed"
**Решение:**
1. Проверьте Wi-Fi соединение
2. Убедитесь, что телефон и компьютер в одной сети
3. Запустите Expo с HTTPS адресом (ngrok): `API_URL="https://<ngrok>.ngrok-free.app" API_KEY="superdev123" npx expo start --tunnel`

---

## 🚀 Режим туннеля (если Wi-Fi не работает)

Если телефон и компьютер в разных сетях, используйте туннель:

```bash
npx expo start --tunnel
```

Это создаст туннель через интернет, и телефон сможет подключиться из любой сети.

**Примечание:** Режим туннеля может быть медленнее, но работает из любой сети.
