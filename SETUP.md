# Инструкция по настройке AdilAI

## Быстрый старт

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка переменных окружения

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Отредактируйте `.env`:
```
API_URL=http://127.0.0.1:8000
API_KEY=superdev123
TENANT_ID=default
NOTIFICATIONS_ENABLED=true
```

### 3. Запуск приложения

```bash
npx expo start
```

## Размещение ресурсов из Figma

Поместите следующие файлы в папку `app/assets/`:

- `icon.png` (1024x1024) - иконка приложения
- `splash.png` (1242x2436) - экран загрузки
- `adaptive-icon.png` (1024x1024) - адаптивная иконка для Android
- `favicon.png` (48x48) - иконка для веб-версии
- `notification-icon.png` (96x96) - иконка для уведомлений

## Обновление локализации

Отредактируйте файлы:
- `app/i18n/ru.json` - русский язык
- `app/i18n/kz.json` - казахский язык

После изменений перезапустите приложение.

## Команды разработки

```bash
# Запуск
npm start

# Линтинг
npm run lint
npm run lint:fix

# Форматирование
npm run format

# Тесты
npm test
```

## Структура проекта

- `app/components/` - переиспользуемые компоненты
- `app/screens/` - экраны приложения
- `app/services/` - API клиенты и утилиты
- `app/store/` - Zustand stores
- `app/theme/` - темы (светлая/тёмная)
- `app/i18n/` - локализация
- `app/navigation/` - навигация

## Подключение к бэкенду

1. По умолчанию клиент подключается к:
   - `http://10.0.2.2:8000` (Android эмулятор)
   - `http://127.0.0.1:8000` (iOS симулятор/Web)
2. Для реального устройства задайте переменные окружения при запуске:
- `API_URL="https://<ngrok>.ngrok-free.app" API_KEY="superdev123" TENANT_ID="default" npx expo start --clear`
3. Для локальной сети можно использовать IP хоста: `API_URL="http://192.168.x.x:8000"`
4. Приложение автоматически использует мок-данные, если сервер недоступен; статус отображается в `HealthBadge`, а дополнительные проверки доступны на экране Debug (Настройки → Debug).

## Известные ограничения

- На iOS может потребоваться ручное предоставление разрешений для камеры
- PDF экспорт на iOS использует системный диалог печати
- WebView для предпросмотра документов может требовать дополнительной настройки
