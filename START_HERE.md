## 📋 БЫСТРЫЙ СТАРТ (3 шага)

### Шаг 1: Откройте терминал в папке проекта

```bash
cd c:\Users\amang\OneDrive\Desktop\adilai
```

### Шаг 2: Запустите сервер

```bash
npx expo start
```

### Шаг 3: Выберите платформу

После запуска появится меню с QR-кодом:

- **Нажмите `w`** → Запустится веб-версия в браузере (самый простой способ!)
- **Нажмите `a`** → Запустится на Android (нужен эмулятор или устройство)
- **Нажмите `i`** → Запустится на iOS (нужен Mac)
- **Отсканируйте QR-код** → Приложением Expo Go на телефоне

## 🔧 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "Port 8081 is being used"
**Решение:** Expo автоматически предложит использовать другой порт (8082). Просто нажмите `yes`.

### Проблема: Ошибки с assets
**Решение:** Это нормально! Приложение работает без иконок. Просто игнорируйте эти предупреждения.

### Проблема: Сервер не запускается
**Решение:** 
```bash
npx expo start --clear
```

### Проблема: Ошибка с package.json
**Решение:** Уже исправлено! Если появится снова после `npm install`, см. `QUICK_START.md`.
